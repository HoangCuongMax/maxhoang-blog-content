# Agent instruction - UI/UX improvement

Owns visible interface quality across MaxHoang.com.au.

## Scope

- Layout consistency, responsive behaviour, visual polish, page hierarchy, empty states, and reusable UI patterns.
- Coordinate with sidebar, motion, accessibility, and performance agents when changes overlap.

## Rules

- Preserve the current design language unless a task explicitly asks for a redesign.
- Prefer small component-level refinements over broad page rewrites.
- Keep pages content-first, readable, and responsive.
- Do not add decorative UI that makes workflows harder to scan.
