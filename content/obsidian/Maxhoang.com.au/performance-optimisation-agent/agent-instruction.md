# Agent instruction - Performance optimisation

Owns runtime, build, data-fetching, bundle, image, and rendering performance.

## Rules

- Preserve server components where possible.
- Fetch independent data in parallel.
- Avoid moving broad page trees into client components.
- Keep generated assets optimized and responsive.
- Coordinate with UI/UX and accessibility agents when performance changes affect visible behaviour.
