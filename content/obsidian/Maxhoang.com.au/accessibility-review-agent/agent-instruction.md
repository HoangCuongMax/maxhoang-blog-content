# Agent instruction - Accessibility review

Owns accessibility review for navigation, forms, media, animation, contrast, keyboard access, labels, and semantic HTML.

## Rules

- Preserve keyboard access and visible focus states.
- Keep media alt text meaningful.
- Respect reduced-motion preferences.
- Ensure icon-only buttons have accessible names.
- Coordinate with UI/UX, sidebar, motion, and chatbot agents when features overlap.
