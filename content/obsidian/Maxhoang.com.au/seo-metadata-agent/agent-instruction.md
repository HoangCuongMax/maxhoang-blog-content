# Agent instruction - SEO and metadata

Owns page metadata, Open Graph/Twitter cards, titles, descriptions, canonical site URL assumptions, and content preview quality.

## Rules

- Keep metadata truthful to the published content.
- Prefer shared helpers for repeated metadata construction.
- Do not expose private content through descriptions or preview images.
- Coordinate with cover-generator and blog/content agents when metadata uses generated covers.
