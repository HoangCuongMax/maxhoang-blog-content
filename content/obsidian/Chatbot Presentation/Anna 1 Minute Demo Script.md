# Anna 1 Minute Demo Script

Purpose: record a short demo that shows Anna as a friendly website guide, content search assistant, navigation helper, and private Admin Assistant entry point.

## Before Recording

- Open the website locally or live.
- Open Anna from the chatbot button.
- If the intro screen appears, click **Skip** or **Start chatting**.
- Keep the chatbot expanded if you want the answers to be easy to read.
- Optional: turn on Anna voice output with the speaker button.

## Conversation Script

### 0-8 seconds - Greeting

**Max:** Hi Anna.

**Anna:** Hi, nice to meet you. I'm here to help you explore Max's projects, AI notes, blog posts, and events.

You can ask me things like "show me AI projects" or "find beginner AI blogs".

### 8-16 seconds - Ability Overview

**Max:** What can you help me with on this website?

**Anna:** Yes, I can guide you. I can help you find Max's projects, explain blog posts, surface AI notes, point to awards or events, and help you contact Max.

### 16-26 seconds - Max's Strengths

**Max:** What is Max good at?

**Anna:** Max is strongest where AI, data, and practical website systems meet. His work covers custom AI assistants, NLP learning, data analysis, SQL dashboards, automation, WebGIS ideas, GitHub-based content workflows, and Next.js websites connected to Markdown knowledge bases.

### 26-36 seconds - NLP and Chatbot Knowledge

**Max:** Can you find something about NLP or chatbot learning?

**Anna:** A good starting point is Max's NLP and chatbot learning content. It covers ideas like tokenisation, chatbot training, question-answer behaviour, retrieval, and how a website assistant can improve by learning from curated Markdown knowledge.

### 36-45 seconds - Service Capability

**Max:** Can Max build custom AI chatbots?

**Anna:** Yes. Max can build custom AI chatbot experiences that connect to real website content, answer visitor questions, use a defined personality, search project knowledge, collect feedback, and support private admin workflows like refreshing content or updating the chatbot knowledge base.

### 45-52 seconds - Visitor Use Case

**Max:** Show me something useful for a visitor.

**Anna:** A visitor can ask me to find Max's AI projects, explain a blog post, recommend where to start learning AI, open the contact page, or show examples of Max's portfolio work. I work like a guided search layer, so visitors do not have to dig through the whole site manually.

### 52-58 seconds - Navigation

**Max:** Open the Daily Journal.

**Anna:** Sure, opening the Daily Journal now.

Expected screen action: the website navigates to `/daily-journal`.

### 58-68 seconds - Admin Assistant Reveal

**Max:** I am Max. Open Admin Assistant.

**Anna / screen action:** Admin Assistant mode opens. If the session is locked, the secure unlock panel appears. If it is already unlocked, the private admin console appears.

Point out the visible admin sections:

- Content Studio / TinaCMS
- Daily Journal creation
- Website cache refresh
- Chatbot knowledge update
- System theme controls

### Optional Closing Line

**Max:** What makes you different from a normal chatbot?

**Anna:** I am different because I am connected to Max's actual website content, notes, blog posts, project knowledge, and private admin workflow. I can answer as a guide, help visitors navigate, show relevant sources, and switch into Admin Assistant mode when Max unlocks private controls.

**Max:** Thanks Anna. That is perfect.

**Anna:** You're welcome. Glad I could help.

## Exact Trigger Lines That Work Best

Use these exact lines if you want the cleanest demo:

1. `Hi Anna.`
2. `What can you help me with on this website?`
3. `What is Max good at?`
4. `Can you find something about NLP or chatbot learning?`

6. `Show me something useful for a visitor.`
7. `Open the Daily Journal.`
8. `I am Max. Open Admin Assistant.`
9. `What makes you different from a normal chatbot?`
10. `Thanks Anna.`

## Demo Notes

- `Open the Daily Journal` works because Anna has a navigation shortcut for Daily Journal.
- `I am Max. Open Admin Assistant` works because Anna recognises admin access phrases and switches into Admin Assistant mode.
- The demo answers for Max's strengths, chatbot services, NLP learning, visitor usefulness, and normal-chatbot comparison are supported by direct answer logic so the recording stays reliable.
- Keep the whole recording around one minute by using only the first eight lines. Use the optional closing line if you have extra time.
