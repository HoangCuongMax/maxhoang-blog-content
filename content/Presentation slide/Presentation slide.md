# Presentation Slide

Create public presentation slide cards in this folder.

Each file represents one presentation card. Use `embedUrl` for the embedded presentation and `viewUrl` for the external button/link.

```yaml
---
title:
slug:
type: presentation
embedUrl:
viewUrl:
author:
buttonLabel: Open embedded presentation
published: true
featured: true
sortOrder: 1
summary:
---
```
